# MiND Study — GABA/GLX Sex Differences Analysis

R analysis examining sex differences in GABA and glutamate+glutamine (GLX) measured via magnetic resonance spectroscopy (MRS) across the adult lifespan, and in the context of mild cognitive impairment (MCI).

**Author:** Molly Simmonite

---

## Overview

MRS data were collected from three groups:

| Group | n (W1) | Age range |
|---|---|---|
| Young cognitively normal (CN) | 60 | ~18–30 |
| Older CN | 170 | ~60–85 |
| Older MCI | 46 | ~60–85 |

**Metabolites:** GABA (G) and GLX, each in uncorrected (UNC) and alpha-tissue-corrected (ATC) forms.

**Voxels:** Auditory (AUD), Sensorimotor (SM), Visual (VV). Exploratory voxels: memory (MEM), lateral parietal (LPV).

**Design:** Cross-sectional at Wave 1; older CN participants have a Wave 2 follow-up for longitudinal analyses.

## Scripts

### Data preparation (run in order)

| Script | Description |
|---|---|
| `01_create_dataset.R` | Load raw data, apply inclusion/exclusion criteria, recode variables |
| `02_MRS_QC_check.R` | Flag implausible MRS values; review output before running 03 |
| `03_MRS_flags.R` | Apply flags, set bad values to NA, apply GM fraction exclusions, save analysis dataset |
| `00_descriptives.R` | Sample characteristics tables |

### Primary analyses

| Script | Description |
|---|---|
| `04_young_sex_differences.R` | Sex differences in young CN (MANOVA + univariates, Age covariate) |
| `05_older_CN_sex_differences.R` | Sex differences in older CN |
| `06_older_CN_longitudinal.R` | Longitudinal change in older CN (Sex × Time mixed models) |
| `07_older_MCI_sex_differences.R` | Sex differences in older MCI |
| `08_CN_vs_MCI_Sex.R` | Sex × Diagnosis interaction (CN vs MCI) |

### Follow-up analyses

The `a` scripts repeat the primary analyses with additional covariates. The `b` scripts add hemisphere as a factor (Sex × Hemisphere interactions using bilateral voxels).

| Script | Description |
|---|---|
| `04a` / `05a` / `06a` / `07a` / `08a` | Primary models + additional covariates |
| `04b` / `05b` / `06b` / `07b` / `08b` | Bilateral hemisphere analyses |

### Standalone scripts

| Script | Description |
|---|---|
| `sensitivity_MoCA_covariate.R` | Sensitivity check: older CN sex differences with MoCA as additional covariate |
| `figures_presentation.R` | Generates presentation figures from primary results |
| `helpers.R` | Shared functions sourced by all analysis scripts |
| `master_run.R` | Runs all scripts in order |

## How to run

Open `analysis.Rproj` in RStudio to set the working directory correctly, then run:

```r
source("scripts/master_run.R")
```

Or run individual scripts in order (data prep first, then analyses).

`sensitivity_MoCA_covariate.R` and `figures_presentation.R` are run separately as needed.

### Dependencies

```r
install.packages(c("tidyverse", "lme4", "lmerTest", "emmeans", "ggplot2"))
```

## Exclusions

- Participants on **Estrogen** or **GABApentin** at a given wave are excluded at the wave level (prior waves retained).
- MRS values are set to NA where the grey matter fraction in the voxel is < 0.2.
- Only CN (subgroup 1) and MCI (subgroup 2) participants are included; uncertain diagnoses and Alzheimer's are excluded.
